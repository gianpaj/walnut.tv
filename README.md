## Getting Started

```bash
browser-sync start --server --files="*.html, js/*.js, css/*.css"
```

### Prerequisites

- Node.JS
- a browser
- YouTube API key to search videos there

### Installing

```bash
npm install browser-sync -g
```

## Deployment

Automatically via Netlify
